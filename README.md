# NESTJS BOILERPLATE

## Description

I like to use NESTJS framework for developing of back-end. But I did not found any usefull boilerplate for it. 
So then I create one...

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
